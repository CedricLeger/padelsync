Aide et questions fréquentes

Fournisseurs d'emails supportés

Padel Sync reconnaît automatiquement les emails de :

  • Arena (livexperience.fr) — Invitations et réservations
  • TenUp / FFT (contact@clients.fft.fr) — Confirmations de réservation
  • DoInSport (doinsport.com) — Confirmations de réservation

Vous pouvez ajouter d'autres expéditeurs dans la configuration. L'add-on essaiera de parser les emails avec ses différents formats.

Questions fréquentes

L'add-on ne détecte pas mes emails

Vérifiez que :

  • L'expéditeur est bien dans votre liste de configuration (Configuration > Expéditeurs).
  • Le trigger de scan est actif (visible sur la page d'accueil de l'add-on).
  • L'email n'a pas déjà le label "PadelSync" (= déjà traité).

Les heures des événements sont décalées

Vérifiez votre fuseau horaire dans la configuration. Il doit correspondre au fuseau de votre lieu de jeu (ex : Europe/Paris, America/Martinique).

Comment retraiter les anciens emails ?

Utilisez la fonction "Supprimer tous les événements" dans l'add-on. Cela retire aussi le label "PadelSync" des emails, ce qui permet au prochain scan de les retraiter.

Pour rattraper un historique important, augmentez temporairement le "Max emails par scan" à 100 ou 200 dans la configuration.

Comment ajouter un nouveau fournisseur ?

Dans la configuration, ajoutez l'adresse email du fournisseur dans le champ "Expéditeurs", séparée par une virgule. Si vous devez filtrer par sujet d'email, utilisez la syntaxe : adresse@email.com|Mot clé dans le sujet

Comment désinstaller Padel Sync ?

  1. Optionnel : utilisez "Supprimer tous les événements" dans l'add-on pour nettoyer votre calendrier.
  2. Allez sur le Google Workspace Marketplace.
  3. Trouvez Padel Sync dans vos applications installées.
  4. Cliquez sur "Désinstaller".

Mes données sont-elles sécurisées ?

Oui. Padel Sync ne stocke aucune donnée en dehors de votre compte Google. Aucune donnée n'est envoyée à des serveurs externes. Consultez notre politique de confidentialité pour plus de détails.
