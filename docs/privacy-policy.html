Politique de confidentialité

Dernière mise à jour : 25 février 2026

En résumé : Padel Sync ne collecte, ne stocke et ne partage aucune donnée personnelle en dehors de votre propre compte Google. Toutes les données restent dans votre environnement Google.

1. Données traitées

Padel Sync accède aux données suivantes de votre compte Google :

  • Emails Gmail : lecture des emails provenant d'expéditeurs de réservation padel que vous avez configurés (ex : Arena, TenUp/FFT, DoInSport). Seuls le sujet, le corps du message et les métadonnées (date, expéditeur) sont lus pour extraire les informations de réservation.
  • Labels Gmail : création et gestion d'un label "PadelSync" pour marquer les emails déjà traités.
  • Google Calendar : création, modification et suppression d'événements dans le calendrier que vous avez choisi.

2. Stockage des données

Padel Sync stocke uniquement :

  • Votre configuration (expéditeurs, fuseau horaire, préférences) dans les User Properties de Google Apps Script, accessibles uniquement par vous.
  • Un historique des synchronisations (date, type, statut — max 50 entrées) dans vos User Properties.

Aucune donnée n'est stockée en dehors de votre compte Google. Il n'y a pas de serveur externe, pas de base de données tierce, pas de stockage cloud autre que Google.

3. Partage des données

Padel Sync ne partage aucune donnée avec des tiers. Spécifiquement :

  • Aucune donnée n'est vendue.
  • Aucune donnée n'est utilisée à des fins publicitaires.
  • Aucune donnée n'est transmise à des services externes.
  • Aucune donnée n'est accessible par le développeur de l'add-on.

4. Utilisation des scopes Google

L'utilisation de Padel Sync et son transfert à tout autre application des informations reçues des API Google respectent la Politique relative aux données utilisateur des services API Google, y compris les exigences d'utilisation limitée (Limited Use).

Les scopes demandés sont :

  • Gmail (modification) : pour lire les emails de réservation et gérer les labels.
  • Calendar : pour créer et gérer les événements padel.
  • Script : pour gérer les triggers de scan automatique.

5. Suppression des données

Pour supprimer toutes les données de Padel Sync :

  • Désinstallez l'add-on depuis le Google Workspace Marketplace. Cela supprime automatiquement tous les triggers et les User Properties associés.
  • Supprimez les événements créés via la fonction "Supprimer tous les événements" dans l'add-on avant de le désinstaller.
  • Le label Gmail "PadelSync" peut être supprimé manuellement dans les paramètres Gmail.

6. Sécurité

  • Toutes les communications passent par les API sécurisées de Google (HTTPS).
  • La configuration de chaque utilisateur est isolée (User Properties).
  • Les entrées utilisateur sont validées (protection contre les injections).
  • Le contenu des emails n'est jamais stocké — seules les informations extraites (date, heure, terrain) sont utilisées.

7. Modifications de cette politique

En cas de modification de cette politique, la date de mise à jour sera actualisée en haut de cette page. Les changements significatifs seront notifiés via les notes de mise à jour de l'add-on sur le Marketplace.
